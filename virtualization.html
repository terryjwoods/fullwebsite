Virtualization

What is virtualization?

Virtualization is a method used to software that will replace the need of buying hardware to run operating systems. This allows users to use more than one operating system on just one computer.

List of Operating Systems available for virtualization:

  • Windows 7
  • Window 2008 server
  • Linux (ubuntu)
  • Mac Os
